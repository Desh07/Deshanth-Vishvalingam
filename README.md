# Deshanth Vishvalingam – Portfolio

A modern, responsive portfolio website built with **React** and **Vite** to showcase my skills, projects, and experience as a Software Quality Assurance Engineer and Freelance Web Developer.

## 🚀 About

This portfolio highlights:

* Software Quality Assurance skills
* Manual Testing
* Test Automation with Selenium
* API Testing
* CI/CD knowledge
* Freelance web development projects
* Education, experience, and contact information

## 🛠️ Built With

* React
* Vite
* JavaScript
* CSS
* Lucide React

## ✨ Features

* Responsive design
* Smooth scrolling navigation
* Interactive UI with animations
* QA Engineer / Freelance Developer mode switch
* Project showcase
* Skills and experience sections
* Contact section

## 📦 Getting Started

Clone the repository:

```bash
git clone https://github.com/Desh07/portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
